<h3> Local Development:  </h3>

**1. Clone the repository**

```bash
git clone https://github.com/your-username/fail-u-forward.git
```

**2. Move into the project directory**

```bash
cd fail-u-forward
```

**3. Install dependencies**

```bash
npm install
```

**4. Start the dev server**

```bash
npm run dev
```
